## 🚀 QUICK START (5 STEPS - 45 MINUTES)

### Step 1: Setup
```bash
bash quick-start.sh
```
**Duration**: 5 minutes
**What it does**: Checks Python/Node.js, installs dependencies

### Step 2: Train Models
```bash
cd backend
python3 train_models.py
```
**Duration**: 30 minutes (first time only)
**What it does**: Trains all 5 models on 65,534 network logs

### Step 3: Verify
```bash
python3 validate_system.py
```
**Duration**: 2 minutes
**What it does**: Checks all components are working

### Step 4: Start Services (3 terminals)

**Terminal 1**:
```bash
ollama serve
```

**Terminal 2**:
```bash
cd backend && python3 app.py
```

**Terminal 3**:
```bash
npm run dev
```

### Step 5: Open Dashboard
```
http://localhost:5173
```

✅ **Done!** System is running!
